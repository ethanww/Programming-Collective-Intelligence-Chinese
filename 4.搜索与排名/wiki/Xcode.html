Xcode is Apple Computer's suite of tools for developing software on Mac OS X, which ships free with that operating system.

The main application of the suite is the integrated development environment (IDE), also named Xcode. The Xcode suite also includes most of Apple's developer documentation, and Interface Builder, an application used to construct graphical interfaces.

The Xcode suite includes the open source GNU Compiler Collection (GCC), and can compile C, C++, Objective C++, Java, AppleScript, and Objective-C source code with a variety of programming models, including but not limited to Cocoa, Carbon, and Java. Third parties have added support for GNU Pascal,[1] Free Pascal[2] and Ada.[3] The Xcode suite uses GDB as the back-end for its debugger.

[edit]

Major features

Among the highly touted features of the Xcode suite is the technology to distribute the building of source code over multiple computers. Xcode has two systems for distributing compiles. The first uses the Bonjour protocol and the open source distcc. The second (and more recently added) Apple calls Dedicated Network Builds, which scales better to larger configurations than distcc but does not support Bonjour's auto-discovery service.

Xcode can build universal binaries which allow Mac OS X software to run on both PowerPC and Intel-based (x86) architectures. Xcode can build 32- and 64-bit PowerPC and 32- and 64-bit Intel code.

Xcode also includes Apple's WebObjects tools and frameworks for building Java web applications and web services (previously sold as a separate $699 product).

In August 2006, Apple announced that DTrace, a dynamic tracing framework created by Sun Microsystems and released as part of OpenSolaris, would be integrated into Xcode.

[edit]

History

Main article: History of Xcode

The Xcode suite, like Mac OS X itself, had its roots in technology from NeXT. Prior to the October 24, 2003 release of Mac OS X v10.3, Apple referred to the suite of tools as Developer Tools. The Xcode IDE was developed from and superseded Apple's earlier IDE, Project Builder.
