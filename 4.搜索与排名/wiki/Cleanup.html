Cleanup

Many cleanup templates listed below need their very own minor cleanup. They add tagged pages to a category, unfortunately also lists with examples. To disable this replace
...<includeonly>[[Category:etc.]]</includeonly> by
...<includeonly>{{{category|[[Category:etc.]]}}}</includeonly>.

Tag Text that will be shown (and usage notes)
{{cleanup}}

category links talk view
To meet Wikipedia's quality standards, this article or section may require cleanup.
Please discuss this issue on the talk page, and/or replace this tag with a more specific message. Editing help is available.
  • This applies to general problems not addressed by other tags.
  • In the future, please use cleanup-date, below, instead.
{{cleanup-date|September 2006}}

category links talk view
To meet Wikipedia's quality standards, this article or section may require cleanup.
Please discuss this issue on the talk page, or replace this tag with a more specific message. Editing help is available.
This article has been tagged since September 2006.
  • This applies to general problems not addressed by other tags.
  • Please do not use this in combination with the more specific cleanup tags.
{{cleanup-remainder|September 2006}}

category links talk view
To meet Wikipedia's quality standards, the remainder of this article may require cleanup.
Please discuss this issue on the talk page, and/or replace this tag with a more specific message. Editing help is available.
This article has been tagged since September 2006.
  • Use this tag where the remaining sections of an article need cleanup.
{{CleanupConfusing|September 2006}}

category links talk view

To meet Wikipedia's quality standards, this article or section may require cleanup to avoid discrepancies and confusing statements. Please discuss this issue on the talk page, or you may attempt to improve this page yourself. This article has been tagged since: September 2006


  • Use this tag in articles that need broad cleanup to eliminate confusion and discrepancies.
{{cleanup-list}}

category links talk view
To meet Wikipedia's quality standards, this list may require cleanup.
This list is poorly defined, unverified or indiscriminate.
If you are familiar with the subject, please improve the list, prune it, or discuss its parameters on the talk page. Editing help is available.
  • Use this tag where a list needs cleanup.
{{cleanup-image}}

category links talk view
To meet Wikipedia's quality standards, this image may require cleanup.
Please see our picture tutorial and image use policy for further information. Image help is available.
  • This should be used for images that have artifacts, poor definition, poor lighting, or other visual flaws.
  • Many of these images may need to be replaced entirely. Try listing on Wikipedia:Image recreation requests.
{{disambig-cleanup}}

category links talk view
This is a disambiguation page: a list of articles associated with the same title. If an internal link referred you to this page, you may wish to change the link to point directly to the intended article.
Note: This page needs to be cleaned up to be brought into conformance with the Manual of Style. See Wikipedia:WikiProject Disambiguation for more information



  • Use this for disambiguation pages not written according to the style in MoS:DP.
  • Replace Template:Disambig (edit talk links history) with this.
{{cleanup-afd}}

category links talk view
To meet Wikipedia's quality standards, this article or section may require cleanup, suggested by its AFD discussion.
Please discuss this issue on the talk page, and/or replace this tag with a more specific message. Editing help is available.
  • Use this tag where an AFD discussion closes with a consensus of "keep and cleanup."

This page may be a user page mistakenly created as an article.

This page doesn't seem to assert the notability of its subject, which means that this page may qualify for deletion as a vanity page. If the creator of this page is a new user, and would like to keep this information posted on Wikipedia, the information in this article can be transferred to the User page of their account.
