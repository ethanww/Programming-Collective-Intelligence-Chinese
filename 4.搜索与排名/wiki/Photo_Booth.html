Photo Booth is a small software application for taking photos with an iSight camera by Apple Computer for Mac OS X. It features 17 built-in effects that can be applied to photos. To provide illumination for the photos taken, the screen flashes white, replicating the flash on a camera. The program is bundled with the iMac G5, iMac (Core Duo), MacBook, and MacBook Pro personal computers, as these systems feature a built-in iSight camera with which photographs can be taken. Photo Booth will also ship in all packages of Apple's upcoming operating system, Mac OS X v10.5 (Leopard).

The software integrates with instant messaging programs such as OS X's built-in iChat Software or Adium, for taking a picture to use as an avatar or Buddy icon.

[edit]

Technical

Photo Booth, when launched, displays the output fed to it by the built-in iSight camera. It features three main functions when in this "Live Input" Mode:

  • Take Picture (a red button with a camera icon below the output screen)
  • Access to set of effects 1
  • Access to set of effects 2

When displaying a taken image, two additional functions are displayed:

  • Send Picture in an email (using Mail)
  • Add Picture to your iPhoto Library.

Pictures taken in Photo Booth sport a standard iSight resolution of 640×480 pixels and image size varies, but stays close to about 100 KB. In Mac OS X Tiger, all images are stored in ~/Pictures/Photo Booth/, where ~ is the logged-in account's home folder.

Photo Booth's technology is taken further in Apple's next iteration of the Mac OS, Mac OS X v10.5 in which users of iChat will be able to apply Photo Booth effects to the live image used by a web cam during a video conference. Further unspecified enhancements have been confirmed to be featured in the upcoming version.

[edit]

Effects

Photo Booth has two sets of image effects that can be applied when taking a picture. The first set may be described to contain photographic filters, familiar from Photoshop:

  • Sepia
  • Black & White
  • Glow
  • Comic Book
  • Normal
  • Colored Pencil
  • Thermal Camera (this is a fake generated image, that works realistically only if the user is light-skinned.)
  • X-Ray
  • Pop Art
Photo Booth's bulge effect
Photo Booth's mirror effect

The second set has been described by Apple's CEO Steve Jobs himself as a set of "Teenage Effects", being a collection of effects that manipulate the shape of the image, rather than filtering it:

  • Bulge
  • Dent
  • Twirl
  • Squeeze
  • Mirror
  • Light Tunnel
  • Fisheye
  • Stretch

Photo Booth also has a feature to automatically flip pictures when taken, as iSight captures a mirrored image.
