JetBrains is a Czech software development company with offices in Prague, Czech Republic; Saint Petersburg, Russia and Boston, USA. It is known for its Java IDE, IntelliJ IDEA.

JetBrains was founded in 2000 as a private company. JetBrains' cofounders are Sergey Dmitriev, Eugene Belyaev and Valentin Kipiatkov.

[edit]

Released Products

  • IntelliJ IDEA - an intelligent Java IDE intensely focused on developer productivity.
  • ReSharper - the refactoring and productivity plugin for Microsoft Visual Studio.NET, supporting C# language
  • Omea - the "Personal Information Environment", integrating all kinds of resources, like files, emails, news messages, web bookmarks, tasks, contacts, etc in one place.
  • dotTrace - an efficient, easy to use CPU profiler for hunting down performance bottlenecks in .NET applications.
  • TMate - a combined IntelliJ IDEA plugin and web application that helps work with CVS and Subversion repositories
[edit]

Products in Development

  • Meta Programming System - a new programming environment which makes it easy to define your own specialized languages, and use them together with any other language. It implements concepts of Language Oriented Programming.
