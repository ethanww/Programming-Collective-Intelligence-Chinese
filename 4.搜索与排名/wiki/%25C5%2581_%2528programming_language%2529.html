Ł (programming language)

Ł (can be spelled L/U alternatively) is a high-level structured imperative stack-based interpreted programming language. It uses of postfix operators (reverse Polish notation), weak typing, pointers, arrays, anonymous functions (some kind of closures, and two stacks - one for function calls, the other one for variables. It was created in 2006. Its development was suspended, leaving only alpha version of interpreter, which was too slow for practical purposes.

[edit]

Examples

A Hello, world! program:

function main :
  "Hello, world!" print
  nl
;

An interactive adding program:

function main :
  input input + print nl
;

An interactive adding program that makes use of variables and modules:

WITH io
var $a $b ;
function main :
  input `$a
  input `$b
  $a $b + print nl
;

An example of conditional statement and anonymous functions:

function main :
  { "positive or 0" }
  { "negative" }
  input 0 >= ifelse
  print nl
;

Preprocessor

WITH includes file from standard location
JOIN includes file from project directory (not recursive - use only in main module)
[edit]

Operators

>< duplicates value on top of stack
<> swaps 2 values on top of stack
<-> swaps 3 values on top of stack
^<< >>^ rotates 3 values on top of stack
pop removes 1st value from stack
pop2 removes 2st value from stack
+ +- -+ * *^ ^* addition, substaction, multiplication and power operators
// \\ integer division operators
/% \% modulo operators
/ \ real division operators
nop no operation
&& || ^^ <==> ==> ! logical operators
& | ^ <=> => ~ (- (/- bitwise and set operators
&3 |3 ^3 <=>3 =>3 ~3 operators for 3-value logic
%& %| %^ %<=> %=> %~ fuzzy logic operators
>- variable pointer dereference
#$a pointer to $a
#+ pointer to + operator
+> <+ concatenation operators
= /= < > <= >= << <=< <<= <=<= relation operators
