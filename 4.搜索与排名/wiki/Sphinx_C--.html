Sphinx C--

From Wikipedia, the free encyclopedia

Jump to: navigation, search

Sphinx C-- is a compiler for a C-- like programming language by Peter Cellik for DOS. The language is a mix between C and the assembly languages and allows combining assembly commands with C-like structures. The goal was to avoid the overhead usually associated with high-level programming languages but keeping their structures and readability. The package included the compiler, an IDE, libraries, example programs and basic documentation. The output of the compiler were COM and OBJ files.

Peter ended the development of C-- in 1996 making the source code of the compiler available to the public.

The SPHiNX Programming Team took over the development and released several versions which fixed bugs and included a Microsoft Windows-based IDE.

The Russian Mishel Sheker also started modifying the source and has enhanced the compiler in many ways:

  • it's now a 32-bit protected mode program using a DOS extender, a Win32 console version also exists
  • outputs EXE, PE, LE, COFF, SYS and MenuetOS files
  • inclusion of FPU, Intel Pentium, Pentium Pro, Pentium II, MMX and SSE opcodes
  • libraries to make compiling programs created for Turbo C possible with minimal modification.

The program is distributed under greenware licence, making it freeware.
