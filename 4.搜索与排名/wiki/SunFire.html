SunFire

From Wikipedia, the free encyclopedia

Jump to: navigation, search

The Sun Fire series is a range of UNIX server computers produced by Sun Microsystems from 2001 onwards, superseding the Sun Enterprise series. Initially, all Sun Fire systems were based on the UltraSPARC 64-bit microprocessor architecture; some later models use Intel IA-32 or AMD64 architecture processors and are essentially PC compatible servers.

UltraSPARC-based Sun Fire models run the Solaris Operating Environment; IA-32 or AMD64-based Sun Fires support Solaris x86 with Red Hat Enterprise Linux, SuSE Linux Enterprise Server, Windows 2000 or Windows Server 2003 as options depending on model.

Later Sun Fire model numbers have prefixes indicating the type of system thus:

  • V: low-end and mid-range servers (UltraSPARC, IA-32 or AMD64)
  • E: enterprise-class servers with high-availability features (UltraSPARC)
  • B: blade servers (UltraSPARC or IA-32)
  • X: rack-mount AMD64-based servers
  • T: servers based on UltraSPARC T-series CoolThreads processors

For a while Sun used an x suffix to denote IA-32 architecture systems and a z suffix for AMD64 architecture systems, but this convention was later dropped. The z suffix was previously also used to differentiate the V880z Visualization Server variant of the V880 server.

All the system below not marked as EOL (End of Life) are still in production.
