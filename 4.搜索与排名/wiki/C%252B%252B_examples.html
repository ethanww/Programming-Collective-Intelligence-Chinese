These are some examples of code from the C++ Programming Language.

[edit]

Conditionals

C++ files include a header, for example:

#include <iostream>
using namespace std;

Here is a simple 'if' statement:

if (tester)
    cout << "true" << endl;

(Note that "tester" is not an actual C++ statement, but shows where a true/false test goes.) "cout" refers to the output. Input is referred to by "cin." Here is an 'if' statement which also has an 'else' clause:

if (tester)
    cout << "true" << endl;
else
    cout << "false" << endl;

You can also combine ifs, else ifs, and elses all in the same statement, such as this code that prints out a grade letter based on percentages:

int grade;
cin >> grade;
if (grade >= 90)
    cout << "A" << endl;
else if (grade >= 80)
    cout << "B" << endl;
else if (grade >= 70)
    cout << "C" << endl;
else if (grade >= 60)
    cout << "D" << endl;
else
    cout << "F" << endl;

The word "grade" in the previous example is a variable to which a number is stored. the first line declares the variable to the computer, and the second line requests an input from the user, which is stored in "grade".

The above code can be written using a 'switch' statement, as follows:

int grade;
cin >> grade;
switch (grade/10)    // uses tens value
{ 
    case 10: cout << "A+!!" << endl; break;
    case 9:  cout << "A"    << endl; break;
    case 8:  cout << "B"    << endl; break;
    case 7:  cout << "C"    << endl; break;
    case 6:  cout << "D"    << endl; break;
    case 5:  case 4: case 3: case 2: case 1: case 0: 
             cout << "F"    << endl; break;
    default: cout << "Invalid grade" << endl;
}

Note that the code which follows an 'if', 'else if', or 'else' is a statement, but it can also be a compound statement with more than one command, which would be written like this :

if (tester)
{
    cout << "true" << endl;
    x=1;
}
else
{
    cout << "false" << endl;
    x=2;
}

The brackets surround the "body" of the test. (x=1 assigns a value to the variable "x").

C++ uses four logic test structures to organize program flow. The previous "if()" and "if()...else" are examples of the one-way and two-way branch, respectively, named for the way they are symbolized in a flowchart. The two other tests are loops: The pre-test loop:

while(x<10) {
    cout << x;
    x=x+1;
}

and the post-test loop:

do {
    cout<<x;
    x=x+1; 
} while (x<10);

STL

The Standard Template Library provides a variety of useful containers for one to use. Here is an example involving the STL vector:

#include <vector>
#include <iostream>

int main()
{
    std::vector<int> my_vector; // Create a vector that will hold integers
    int input = 1;              // Create a temporary variable to hold the user's input

    std::cout << "Enter some numbers, enter zero to quit!" << std::endl;
    while (input) // While input is nonzero
    {
        std::cin >> input;
        my_vector.push_back(input); // This function adds input to the back of the vector
    }
    std::cout << "You entered the following numbers:" << std::endl;

    // Now, iterate through the contents of the vector
    for (std::vector<int>::iterator i = my_vector.begin(); // Vector.begin() returns an iterator
                                                           // pointing to beginning of the list
        i != my_vector.end(); ++i)
    {
        // Use (*my_iterator) to access the contents of the iterator
        std::cout << (*i) << std::endl;
    }
    std::cout << "Goodbye!" << std::endl;
}

Classes

C++ supports classes, which are objects which one may define.

class my_class                                        // class name
{
public:                                               // the following members are
                                                      // publicly accessible
    my_class(const int& data) : m_Data(data) { ... }  // constructor
    virtual ~my_class() { ... }                       // destructor

    int get_data() const { return m_Data; }           // public member function or method

protected:                                            // the following members are accessible
                                                      // only by this class or classes inherited
                                                      // from it
    int m_Data;                                       // private data member

private:                                              // the following members are accessible
                                                      // only by this class, and NOT classes inherited
                                                      // from it
};
