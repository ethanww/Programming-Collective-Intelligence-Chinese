VisSim

From Wikipedia, the free encyclopedia

Jump to: navigation, search

VisSim is a visual block diagram language for modeling, simulating and analyzing dynamic systems. It is developed by Visual Solutions.

VisSim is widely used in control system design and digital signal processing for multidomain simulation and design. It includes blocks for arithmetic, Boolean, and transcendental functions, as well as digital filters, transfer functions, numerical integration and interactive plotting. The free VisSim Viewer lets anyone run VisSim diagrams. Coupled with VisSim/C-Code, an add-on product, VisSim performs code generation for real-time implementation of embedded systems. It can target small 16-bit fixed point systems like the Texas Instruments MSP430 (using only 170 bytes of data RAM) as well as larger 32-bit floating point processors like Texas Instruments C6713. This technique of simulating system performance off-line, and then generating code automatically from the simulated diagram to run on the embedded system is known as "model based development". Model based development is becoming widely adopted for production systems because it shortens development cycles.

[edit]

VisSim add-ons Include:

  • Physical layer communication system simulation
  • Frequency domain analysis (Bode plot, Root locus, Nyquist plot)
  • C programming language code generation
  • Neural networks
  • OPC (OLE for process control) communication
  • Global optimization of system parameters
  • Real-time analog signal and digital I/O under Windows
  • Bit true fixed-point arithmetic blockset for simulation and code generation
  • Code generation for on-chip peripherals like serial ports, CAN, PWM, QEP, Event Capture, SPI, ADC, DAC, and digital I/O
  • Electric motor simulation library for AC induction, Brushless DC, Stepper and Brushless DC motors
