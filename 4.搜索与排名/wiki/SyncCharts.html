SyncCharts

From Wikipedia, the free encyclopedia

Jump to: navigation, search

SyncCharts is a graphical formalism dedicated to reactive system modeling. It allows specifying the reactive behavior, and the synchronous programming, of applications. SyncCharts is the name of the model. A syncChart is an instance.

SyncCharts inherits many features from StateCharts, and Argos. A commercial distribution is in Esterel Studio, from Esterel Technologies. Any syncChart can be automatically translated into an Esterel program, so users can use the Esterel software environment developed for synchronous programming.
