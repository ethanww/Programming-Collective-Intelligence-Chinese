Delimiters

From Wikipedia, the free encyclopedia

Jump to: navigation, search

Delimiters are objects that enclose text or code. Most commonly found in programming languages, such as C++ and PHP, they define something about the code that they enclose.

[edit]

Examples of Delimiters

Listed below are some of the more common delimiters. Some are common to almost all coding, whilst others are more specific.

  • ( and )
  • { and }
  • [ and ]
  • < and >
  • " and "
  • ' and '
  • <? and ?>
  • /* and */
  • {| and |}
[edit]

Uses

Delimiters are used to define an attribute of the enclosed code or text. They are used in almost all coding languages, including Wikicode, and the attribute they define can be almost anything. For example, the delimiters /* and */ and commonly recognised as comments. The code that they surround is ignored by compilers and is only there to remind the programmer about something or to provide useful information to other readers. The attribute that is defined is commented code.
