GNU Mach

From Wikipedia, the free encyclopedia

Jump to: navigation, search
 Free software Portal

GNU Mach is an implementation of the Mach microkernel. It is the default microkernel in the GNU Hurd kernel of the GNU operating system. GNU Mach runs on IA-32 machines, and is expected to be ported to other computers. GNU Mach is maintained by developers on the GNU project. Anybody can use, modify, and redistribute it under the terms of the GNU General Public License (GPL).

Mach is particularly well suited for SMP and network cluster techniques. Thread support is provided at the kernel level, and the kernel itself takes advantage of that. Network transparency at the IPC level makes resources of the system available across machine boundaries. The latest version, GNU Mach 1.3, was released in May 2002 and features advanced boot script support, support for disks larger than 10 gigabytes and an improved console.
