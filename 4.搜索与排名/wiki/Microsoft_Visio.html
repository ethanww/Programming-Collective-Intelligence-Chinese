Microsoft Visio

Microsoft Visio 2003
Maintainer: Microsoft
Latest release: 2003 SP2 / 27 September 2005
OS: Microsoft Windows
Use: Diagramming
License: Proprietary
Website: www.visio.com

Microsoft Visio is diagramming software for Microsoft Windows that is part of the Microsoft Office series of applications. It uses vector graphics to create diagrams.

Visio became a Microsoft product when Microsoft acquired the Visio Corporation in 2000. Visio is now part of the Microsoft Office family of products; however, it is not included in any of the Microsoft Office software packages. It is only available for Windows. Visio 2007, along with Office 2007, are still Beta versions but should be released sometime in the late third or fourth quarter of 2006.
