List of Soviet computer systems

From Wikipedia, the free encyclopedia

Jump to: navigation, search

This is the list of Soviet computer systems. The Russian abbreviation EVM (ЭВМ), present in some of the names below, means "электронная вычислительная машина", literally "electronic computing machine". See History of computer hardware in Soviet Bloc countries for more.

This list is incomplete; you can help by expanding it.
[edit]

Hardware

  • Agat (Агат) — Apple II clone
  • Aragats (Арагац)
  • BESM (БЭСМ)
  • Besta (Беста) — unix box, Motorola 68020-based, Sun-3 clone
  • Dnepr (Днепр)
  • Dubna 48K — improved ZX Spectrum clone
  • Elbrus (Эльбрус)
  • Elektronika (Электроника) family
    • DVK family (ДВК)
    • Electronika BK-0010 (БК-0010, БК-0011)
    • UKNC (УКНЦ) — educational, PDP11-like
    • Electronika 60, Electronika 100
    • Electronika SS BIS (Электроника СС БИС) — Cray clone
  • ES EVM (ЕС ЭВМ), IBM mainframe clone
  • ES PEVM (ЕС ПЭВМ), IBM PC compatible
  • Hobbit (improved ZX Spectrum clone)
  • Iskra (Искра)
  • Irisha (Ириша)
  • Kiev (Киев)
  • KVM-1 (КВМ-1)
  • Korvet (Корвет)
  • M-20, M-220
  • MESM (МЭСМ)
  • Micro-80 (Микро-80)
  • Microsha (Микроша)
  • Minsk (Минск)
  • Mir, Мир (uk:ЕОМ "МИР-1", uk:ЕОМ "МИР-2")
  • Nairi (Наири)
  • Orion-128 (Орион-128)
  • Pentagon (improved ZX Spectrum clone)
  • Poisk (Поиск) — IBM PC XT clone
  • Promin (Проминь)
  • Razdan (Раздан)
  • Radio-86RK (Радио 86РК)
  • Scorpion (improved ZX Spectrum clone)
  • Setun (Сетунь)
  • SM EVM (СМ ЭВМ)
  • Sneg [1] (Снег)
  • Specialist (Специалист)
  • Strela (Стрела)
  • SVS
  • TsUM-1 (ЦУМ-1)
  • UM (УМ)
  • Ural (Урал)
  • Vector-06C (Вектор-06Ц)
  • Vesna [2]
[edit]

Operating systems

  • DEMOS ("Dialogovaya Edinaya Mobilnaya Operatsionnaya Sistema")
  • For BESM
    • D-68 (Д-68, Диспетчер-68, Dispatcher-68)
    • DISPAK ("Dispetcher Paketov", Packet Dispatcher)
    • DUBNA
  • INMOS, (ИНМОС - Интерактивная Мобильная Операционная Система), unix-like
  • MISS ("Multipurpose Interactive timeSharing System"), unix-like
  • For ES EVM
    • DOS/ES ("Disk Operation system for ES EVM")
    • OS/ES ("Disk Operation system for ES EVM")
  • RAFOS (РАФОС), FODOS (ФОДОС) - RT-11 clones
