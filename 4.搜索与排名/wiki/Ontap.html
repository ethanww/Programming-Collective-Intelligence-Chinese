onTap

"Features Without Fixtures"
Maintainer: S. Isaac Dealey
Latest release: 2.4b / March 18, 2006
Use: Framework
Licence: Mostly BSD
Website: fusiontap.com
Prerequisites
Web server
ColdFusion MX

The onTap framework is an open-source framework for ColdFusion which in addition to providing an MVC controller like other ColdFusion frameworks also provides a wide array of feature-rich APIs for rapid application development, including true SQL language abstraction, form management and i18n internationalization features. The onTap framework is available under an OpenBSD style license.

[edit]

Disambiguation

Data ONTAP is the operating system which drives devices manufactured by Network Appliance.
