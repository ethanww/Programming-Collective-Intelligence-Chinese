Trademarks

  • Microsoft
  • Windows including Windows XP, Windows Me, Windows NT
  • Microsoft Disk Operating System (MS-DOS)
  • DirectX - Windows library for audio, user input, and 2D/3D graphics (software)
  • Xbox - Game Console
  • Microsoft Office and Microsoft Works The world's most widespread office suite.
  • MSN TV - Digital Cable TV Topbox
  • Virtual PC - PC virtualization software acquired from Connectix
  • Microsoft SQL Server - Database server
  • Microsoft Exchange server - Email / groupware server
  • MSN - an ISP
  • Microsoft Zune - a future digital music player, similar to Apple Computer's iPod

For a more complete list see Microsoft's website.
