FreeVMS
OS family: VMS
Source model: Open source
Latest stable release: 0.2.13 / September 4, 2006
License: GPL
Working state: Current

FreeVMS is a free software clone —licensed under the terms of the GPL— of the computer operating system VMS. As of 2006, the project was very much in the early stages of development. The latest version is 0.2.13.

It consists of a POSIX kernel and a DCL command line interpreter.
