VVVV

From Wikipedia, the free encyclopedia

Jump to: navigation, search

VVVV is a graphical programming language similar to Pure Data or Max but less focused on audio. Its main feature is hardware accelerated 3d graphics, and connecting physical devices like sensors and controllers.

The user interface is comfortable. One need not know every object by name to start. Typing a letter into any empty object gets a pull down list of all objects starting with that letter. Right clicking on an empty object gets the full object list, sorted by name or category. Hovering over a pin gets a tooltip with the pin name and current value.

VVVV runs with a certain framerate, this means that every node in the dataflow graph is calculated once per frame.

[edit]

Features

  • DirectX based graphics rendering
  • realtime shader development
  • clustering of multiple computers
  • every object can handle arrays/lists (called spreads)
  • dynamic mesh creation
  • unlimited undo (per run)
  • multi screen support
  • increment time mode for non realtime rendering
