Microsoft Money

From Wikipedia, the free encyclopedia

Jump to: navigation, search
Microsoft Money 2006 Premium

Microsoft Money is Microsoft's personal finance software for computers using the Microsoft Windows operating system. A version is also available for Windows Mobile. The latest version, released in July 2006, is called Microsoft Money 2007 and has four editions: Essentials, Deluxe, Premium and Home & Business.

There are versions for the U.S., UK, France, Japan, Canada and an International English edition for other English speaking countries.

The publicly available FAQ for Microsoft Money is available at money.mvps.org.
