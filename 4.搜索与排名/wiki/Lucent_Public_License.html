Lucent Public License

From Wikipedia, the free encyclopedia

Jump to: navigation, search

The Lucent Public License is an open-source license created by Lucent Technologies. It has been released in two versions: Version 1.0 and 1.02.

While the Lucent Public License is not one of the more popular open-source licenses, a number of products have been released under it. It has been approved by the Open Source Initiative. The Free Software Foundation states[1] that it is free software but incompatible with the GNU General Public License.

It is the license used by Plan 9 from Bell Labs.
