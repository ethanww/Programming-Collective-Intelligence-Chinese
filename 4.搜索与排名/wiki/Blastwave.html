Blastwave

From Wikipedia, the free encyclopedia

Jump to: navigation, search

Blastwave.org is an organisation which prepares and packages open source software for the Sun Microsystems' Solaris operating system.

The Blastwave project is the home of CSW or "Community Software for Solaris". The objective of the project is to allow Solaris users to build software in accordance with specified standards and then to provide the SVR4 compliant software packages to other Solaris users. This process includes a testing stage, open evaluation and bug reporting and final release to a "unstable" and "stable" tree much like the Debian project.

The Blastwave build stack is a set of servers that house the build environment. This stack allows members of the Solaris Community to gain access to both tools and expertise such that they may build software in accordance with standards. To date there are some 1,300 software packages for both the Sparc and x86 architectures that support Solaris. The software is built to support Solaris 8 and runs on Solaris 9 and 10. Solaris Express as well as Solaris Express Community Edition are not officially supported however the CSW software set is reported to work well. It should be noted that the older sun4m Sun hardware architecture is officially supported as well as the latest generation Sun UltraSparc hardware and multi-core Opteron implementations.

The CSW software set from Blastwave is available for free to educational institutions as well as non-profit centers and developers. Corporations are encouraged to purchase a yearly software subscription via the Blastwave DVD. All funds go towards the long term operation of the project as well as towards related open source sites (genunix and OPN4) and initiatives. The software on the mirrors sites as well as the Blastwave DVD is best installed via the pkg-get utility. The pkg-get utility operates much like the Debian project apt-get in that applications are fetched from a software source and then a dependency tree is traversed also. This ensures that a complete software suite may be installed with a single simple command line.
