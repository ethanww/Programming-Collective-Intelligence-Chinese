Just BASIC

From Wikipedia, the free encyclopedia

Jump to: navigation, search
JustBASIC Screenshot

Just BASIC is a dialect of the highly popular BASIC programming language of the 1970's for 32-bit Windows computer systems. It's the freeware sister project to Liberty BASIC, popular since 1995. It allows you to create programs with a comprehensive tutorial, help files and a message board.

[edit]

Syntax

As the title suggests and most BASIC dialects are, the syntax is very simple. Below is a list of some simple commands:

  • print - Outputs text to the screen
  • GOTO - Instructs the computer to jump to another point in the computer program, specified by a label.
  • let - Allows the programmer to make one variable equal to another or do mathematical calculations
[edit]

The GUI interface

Just BASIC also allows for the construction of GUI interfaces. It also comes with Freeform-J a WYSIWYG editor for just this purpose.

[edit]

Compiling

Just BASIC automatically compiles the active program on running. To distribute a program, the programmer must tokenize a program, which compiles the active program into bytecode. The tokenized file is executed by the included runtime engine, which is named jbrun101.exe in the current release.

[edit]

Difference between Liberty BASIC and Just BASIC

Here are some differences between Just BASIC and Liberty BASIC:

  • Liberty BASIC supports API calls
  • Liberty BASIC has custom dialogs for fonts, colors, and the printer
  • Liberty BASIC has a POPUPMENU that simulates a right-click menu
  • Liberty BASIC has an internal SORT command
  • Liberty BASIC costs money
  • Liberty BASIC's IDE has a "quick jump" feature that lets you quickly jump to a label or function

The dowload size is about 2.5MB.
